# Customer-Churn-Prediction-and-Analysis
Customer Churn Prediction &amp; Analysis project built as part of the DEPI program. Includes complete data preprocessing, statistical analysis, feature engineering, model training, and API deployment. Uses Python, pandas, scikit-learn, XGBoost, SciPy, and FastAPI to build a high-performance churn prediction model (ROC-AUC ~0.997).
